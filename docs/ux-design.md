## CLI Interface

### Subcommands

- `ut61eplus list` — enumerate connected devices
- `ut61eplus info` — connect to device and print device info
- `ut61eplus read` — continuous measurement reading
- `ut61eplus command` — send button press to meter
- `ut61eplus debug` — raw hex dump mode
